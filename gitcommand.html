Sure, here's a list of 50 Git commands along with a brief description of each: 1. `git init`: Initialize a new Git repository. 2. `git clone `: Clone a repository into a new directory. 3. `git add `: Add changes in the working directory to the staging area. 4. `git commit -m "message"`: Commit changes from the staging area to the repository. 5. `git status`: Show the status of changes as untracked, modified, or staged. 6. `git diff`: Show the differences between the working directory and the staging area. 7. `git log`: Show the commit history. 8. `git branch`: List all branches in the repository. 9. `git branch `: Create a new branch. 10. `git checkout `: Switch to a different branch. 11. `git merge `: Merge changes from one branch into the current branch. 12. `git remote add origin `: Add a remote repository. 13. `git pull origin `: Fetch changes from a remote repository and merge them. 14. `git push origin `: Push changes to a remote repository. 15. `git fetch`: Fetch changes from a remote repository without merging. 16. `git reset `: Unstage changes for a file. 17. `git reset --hard`: Discard all changes in the working directory and staging area. 18. `git rm `: Remove a file from both the working directory and the staging area. 19. `git mv `: Rename a file. 20. `git show `: Show the details of a specific commit. 21. `git tag `: Create a lightweight tag at the current commit. 22. `git tag -a -m "message"`: Create an annotated tag. 23. `git fetch --tags`: Fetch tags from a remote repository. 24. `git push --tags`: Push tags to a remote repository. 25. `git branch -d `: Delete a branch. 26. `git branch -m `: Rename a branch. 27. `git remote -v`: Show remote repositories along with their URLs. 28. `git remote show `: Show information about a remote repository. 29. `git log --graph --oneline --all`: Show a concise graphical representation of the commit history. 30. `git config --global user.name "Your Name"`: Set your global username. 31. `git config --global user.email "your.email@example.com"`: Set your global email address. 32. `git stash`: Save changes in the working directory without committing them. 33. `git stash apply`: Apply the most recent stash. 34. `git cherry-pick `: Apply changes from a specific commit to the current branch. 35. `git rebase `: Reapply commits on top of another branch. 36. `git log --grep=""`: Show the commit history that matches a specific pattern. 37. `git blame `: Show who last modified each line of a file. 38. `git bisect start`: Start a binary search to find the commit that introduced a bug. 39. `git fsck`: Check the integrity of the Git repository. 40. `git remote rename `: Rename a remote. 41. `git submodule add `: Add a submodule to the repository. 42. `git clean -n`: Show which files will be removed by `git clean`. 43. `git reflog`: Show a log of changes to the repository's HEAD. 44. `git remote prune origin`: Remove remote branches that no longer exist on the remote repository. 45. `git log --pretty=format:"%h - %an, %ar : %s"`: Customize the format of the commit log. 46. `git log --since="2 weeks ago"`: Show the commit log since a specific date. 47. `git log --grep="" --oneline`: Show one-line commit messages matching a pattern. 48. `git log -p`: Show the patch introduced by each commit. 49. `git log --graph --decorate --oneline`: Show a compact and decorated commit history. 50. `git remote set-url origin `: Change the URL of a remote repository.
